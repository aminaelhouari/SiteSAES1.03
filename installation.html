Protocole d'Installation

Documentation technique de la mise en place de l'environnement de développement sous Linux Ubuntu.

VirtualBox Ubuntu Apache2
1

Configuration de la machine virtuel

Nous avons décidé d'uilisé Oracle VirtualBox pour la suite de l'installation car c'est un logiciel OpenSource très facile d'utilisation

Choix des ressources

Nous avons configuré une nouvelle machine virtuelle nommée Ubuntu avec les caractéristiques suivantes pour garantir une bonne fluidité :

  • Mémoire RAM 4096 Mo (4 Go)
  • Processeurs 4 Cœurs
  • Disque Dur 80 Go
4 Go de RAM et 4 cœurs permettent de faire tourner l'interface graphique sans faire ralentir l'ordinateur hôte
2

Installation de l'OS (Ubuntu)

Après lancement de la VM nous avons choisi la langue française avec le clavier AZERTY pour l'installation de l'OS (Ubuntu)

Compte Administrateur
Nom complet admin_Gr106_Trio3
Nom de l'ordinateur Ubuntu-VM
Nom d'utilisateur (crée automatiquement) admin-gr106-trio3

On va ensuite redémarrer la Machine Virtuelle

3

Ajouts des additions invité

On va lancer un terminal depuis le CD des additions invités (Pour ça il faut cliquer sur Périphériques -> Insérer l’image CD des Additions invité…)

Commandes exécutées :
admin@Ubuntu-VM:~$ sh autorun.sh.
#Ça va nous demander de nous authentifier donc on remet le mot de passe.
admin@Ubuntu-VM:~$ sudo apt install bzip2 tar open-vm-tools
# On écrit le mot de passe et on attend que ça finisse de charger puis on écrit o
admin@Ubuntu-VM:~$ sudo reboot
Un redémarrage (sudo reboot) a été nécessaire pour appliquer les changements
4

Création des Utilisateurs et et d'un groupe

Conformément au sujet, nous avons créé un groupe de travail et ajouté les utilisateurs membres du trinôme.

  • Groupe créé : Trio3
  • Utilisateurs : Trio3_1, Trio3_2, Trio3_3

On a ensuite créer le groupe

sudo groupadd Trio3
sudo usermod –G Trio3 Trio3_1
sudo usermod -G Trio3 Trio3_2
sudo usermod –G Trio3 Trio3_3
5

Personnalisation & Serveur Web

Personnalisation du Prompt (Invite de commande)

Nous avons modifié le fichier .bashrc pour afficher le nom de l'utilisateur dans le terminal.

Pour ça on a écrit dans le terminal

nano ./.bashrc

nano est un éditeur de texte qui fonctionne dans le terminal directement. ./.bashrc c'est le fichier qu'on va ouvrir ( . signifie que c'est un fichier caché)

echo "----------------"
echo "Bonjour $USER"
echo "----------------"
Installation du Serveur Web Apache

Installation du paquet via la commande sudo apt install apache2. Le serveur est fonctionnel et accessible via l'adresse localhost.
